# Mocha

Mocha Test

## Install Mocha

```js
npm install mocha chai --save-dev
```

Mocha: The test framework.
Chai: A popular assertion library for use with Mocha.

## Run the tests:

```js
npx mocha tests/mathUtils.test.js
```

-

## Run all tests

```js
npx mocha tests/**/*.test.js
```

## Run all tests with reporting

npm i --save-dev nyc

npm run coverage
